## 微信小程序常用组件合集

#### header组件

##### 1.使用
```
<header hasBack="{{true}}">首页</header>
```
##### 2.属性列表
| 属性   |      类型      |  默认值 |
|----------|:-------------:|------:|
| color |  String | #fff |
| background |    String   |   #333 |
| hasBack | Boolean |    false |
| hasGoHome | Boolean |    false |
